# Image Folder

## How to Add Your Images

1. **Save your AI-generated images in this folder:**
   - Path: `c:\Flatline-web2.0\public\images\`

2. **Recommended file names:**
   - `tactical-response.jpg` or `tactical-response.png` (1200 x 800 pixels)
   - `team-briefing.jpg` or `team-briefing.png` (600 x 400 pixels)

3. **After adding your images:**
   - The images will be accessible in your code using: `/images/your-filename.jpg`
   - For example: `/images/tactical-response.jpg`

## Current Images Needed:
- **Tactical Response** - 1200 x 800 pixels (Landscape)
- **Team Briefing** - 600 x 400 pixels (Landscape)

## Supported Formats:
- `.jpg` / `.jpeg` (recommended for photos)
- `.png` (if you need transparency)
- `.webp` (for smaller file sizes)
